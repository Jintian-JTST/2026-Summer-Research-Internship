\section{引言}

标准模型（Standard Model）是现代物理学迄今为止最成功的理论之一\cite{BOSTON2023}，然而它并非完美无缺：在预测缪子的反常磁矩方面，标准模型的理论计算与实验测量之间存在着显著的差异，这被称为缪子 $g-2$ 问题。在美国费米实验室，缪子反常磁矩的精确测量实验已经进行多年，2021年测量结果显示了与标准模型预测之间的约4.2倍标准差的差距\cite{Li2022Muon}。因此，该实验引发了物理学界的广泛关注和讨论，分析该实验数据的拟合过程对于理解这一现象至关重要。

这个程序旨在模拟缪子 $g-2$ 实验中的数据生成和分析过程，从模拟数据中提取物理参数，并通过拟合来验证理论模型。

\section{原理与实现}

实验中，极化的缪子被注入一个均匀的磁场中，其圆周运动的频率记作 $\omega_c$，而缪子自旋的进动频率记作 $\omega_s$。缪子的反常磁矩会导致 $\omega_s$ 与$\omega_c$ 之间存在一个差值，计算可得\cite{PhysRevA103}
\begin{equation}
\omega_a = \omega_s - \omega_c= - a_\mu \frac{q}{m_\mu} B
\end{equation}

其中 $a_\mu$ 是缪子的反常磁矩，定义为\cite{BOSTON2023} $a_\mu = \frac{g-2}{2}$。$q$ 是缪子的电荷，$m_\mu$ 是缪子的质量，$B$ 是磁场强度。通过测量 $\omega_a$ 与 $B$，我们可以间接地测量 $a_\mu$，从而检验标准模型的预测。在真实实验中，磁场强度 $B$ 可以通过质子在储能环中的拉莫进动频率 $\omega_p$ 来实现\cite{Chu2021Muon}，因此 $\omega_a$ 的测量是整个实验的核心。


\subsection{正电子能量分布与计算}

正缪子衰变为一个正电子和两个中微子 $\mu^+ \longmapsto e^+ + \nu_\mu + \bar{\nu}_e$ ，正电子的能量分布由衰变动力学决定。对于极化缪子，正电子概率密度函数由 Michel Spectrum 描述\cite{Fetscher2019MuonDecay}：
\begin{equation}
\frac{d^2 \Gamma}{dx\, d\cos\alpha}
=
\frac{G_F^2 m_\mu^5}{192\pi^3}
\left[
3 - 2x \pm P_\mu \cos\alpha (2x - 1)
\right] \cdot x^2
\end{equation}

等式左侧为单位能量分数 $x = {E_e}/{E_{max}}$ 和极化角（正电子射出方向与缪子自旋方向的夹角） $\alpha$ 的微分衰变率，$G_F$ 是费米常数，$m_\mu$ 是缪子质量，$P_\mu$ 是缪子极化度。式子中的 $\pm$ 取正，因为实验中使用的是正缪子。记 $\cos \alpha = \Omega$，简化后正电子的能量分布可以表示为：
\begin{equation}
f(x,\Omega) = x^2 [3 - 2x + P_\mu \Omega (2x - 1)]
\end{equation}

考察缪子衰变的时刻。记缪子参考系为 $\Gamma'$，在该参考系中缪子处于静止状态。以缪子前进方向（动量 $\mathbf{p}_\mu$ 方向）为 $z$ 轴建立球坐标系，且令缪子动量在 $x$-$y$ 平面内的投影为初始轴线。则正电子的动量方向 $\mathbf{p}_e$ 由极角 $\theta$ 和方位角 $\phi$ 描述。此时，记缪子的自旋方向 $\mathbf{S}$ 与 $z$ 轴的夹角为 $\psi$。该角度可以由缪子相位计算得到，在代码中通过 $\psi = \omega_a T + \phi_0$ 计算，其中 $T$ 是缪子存活时间（即缪子产生到此刻的时间），$\phi_0$ 是初始相位。则
\begin{equation}
\Omega = \cos\alpha = \mathbf{S} \cdot \mathbf{p}_e = \sin\psi \sin\theta \cos\phi + \cos\psi \cos\theta
\end{equation}

\begin{figure}[htbp]
\centering
\includegraphics[width=0.8\textwidth]{MuonPositronDiagram_ManimCE_v0.20.1.png}
\caption{缪子衰变示意图，左图为缪子衰变前状态，右图为缪子衰变后正电子状态。图中 $\mathbf{S}$ 是缪子的自旋，$\mathbf{p}_e$ 是正电子的动量。}
\end{figure}


正电子在 $\Gamma'$ 中可以获得的最高能量 $E'_{max} = (m_\mu^2 + m_e^2) / (2 m_\mu)$。 此时正电子能量为 $E'=x E'_{max}$，动量为 $p'=\sqrt{E'^2 - m_e^2}$。根据狭义相对论，正电子在 $\Gamma$ 中的能量 $E$ 可以通过以下公式计算\cite{Chu2021Muon}：
\begin{equation}
E = \gamma (E' + \beta p' \cos\theta )
\end{equation}


在实验中，我们设定了一个能量阈值 $E_{th}$，只有当正电子的能量 $E_e$ 满足 $E_e > E_{th}$ 时，才会被计数为一个事件。程序中通过比较计算得到的 $E$ 与 $E_{th}$ 来筛选事件，并将满足条件的事件记录下来用于后续分析。

\section{模拟数据拟合与分析}

\subsection{五参数模型}

在全部模拟事件中，我将满足能量阈值 $E_{th}$ 的事件筛选出来。当时间大于 $300\, \mathrm{\mu s}$ 时，因为数据样本不足的缘故，事件数目趋近于零，因此我将时间范围限制在 $[0, 300]\, \mathrm{\mu s}$ 内进行分析。对于该时间范围，切成 $n=5000$ 个时间区间，统计每个时间区间内满足条件的事件数目。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{wiggle_line_fit_plot.png}
\caption{正电子数目随时间变化图像。横坐标为时间 $T/\mathrm{\mu s}$，纵坐标为正电子数目。其中$T\in[0, 300]\, \mathrm{\mu s}$，$n=5000$，$\Delta t = 0.06\, \mathrm{\mu s}$。}
\label{wiggle_line_fit_plot}
\end{figure}

自图像可以非常清晰地看到，正电子数目随时间呈现出明显的振荡(wiggle)现象。该图像可以用一个五参数函数\cite{Fienberg2019}进行拟合：
\begin{equation}
  N(t) = N_0 e^{-t/\tau} \left(1 + A(E)\cos(\omega_a t + \phi_0)\right)
\end{equation}

五个需要拟合的参数分别为：初始事件数目 $N_0$，衰减时间常数 $\tau$，振荡幅度 $A$，反常自旋进动频率 $\omega_a$，以及初始相位 $\phi_0$。拟合结果为 $\omega_a = 1.49302(2)\,\mathrm{rad\ \mu s^{-1}}$，$\phi_0 = 0.008(2)\,\mathrm{rad}$。注意到 $\omega_a$ 的注入参数与拟合参数的偏差在 $1\sigma$ 以内，表明五参数模型能够很好地描述数据。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{wiggle_line_fit_plot_with_fit.png}
\caption{图\ref{wiggle_line_fit_plot}的拟合结果展示。红色曲线为拟合结果，蓝色点为生成数据。}
\end{figure}

在相关文献中\cite{PhysRevD.103.072002,Chu2021Muon,PhysRevLett.131.161802}，常使用摆动图（wiggle plot）来展示超过能量阈值的正电子计数随时间的振荡行为。通过将时间对一个固定周期取模，可以将长时间范围内的数据折叠，从而更清晰地展示振荡结构。

\subsection{增大时间区间的拟合表现}

在真实实验数据的分析中可以观察到，摆动图通常覆盖多个振荡周期，其有效时间范围约为 $600\text{--}700\,\mathrm{\mu s}$\cite{Fienberg2019}。为检验拟合结果对时间窗口的稳定性，我将分析时间范围扩展至 $600\,\mathrm{\mu s}$，并将时间分箱数增加至 $n=10000$ 进行重新拟合，得到 $\omega_a = 1.49302(2)\,\mathrm{rad\ \mu s^{-1}}$。这说明拟合结果对时间窗口的稳定性较好。然而，绘制得到的摆动图在视觉上出现了明显差异。

\centering
\begin{minipage}{0.49\textwidth}
    \centering

\end{minipage}
\hfill
\begin{minipage}{0.49\textwidth}
    \centering
    \includegraphics[width=\textwidth]{600wiggle_plot_time_mod_100_with_fit.png}

\caption{选取$T\in[0, 600]\, \mathrm{\mu s}$，$n=10000$，$\Delta t = 0.06\, \mathrm{\mu s}$在摆动图上的呈现。左图为原始数据，右图为加入拟合函数后的结果。}
\end{figure}

在图像底部，原本清晰的振荡结构逐渐退化为离散的散点分布。这主要是由于缪子衰变概率随着时间增加而指数减小，导致单个时间分箱内满足能量阈值的正电子数显著减少。在对数坐标下，这种离散计数会表现为明显的分层结构，从而形成条带状分布。

\subsection{残差频谱绘制}

拟合完成后，为了研究模型未能描述的时间结构，我构造了数据的绝对残差\cite{BOSTON2023} $r_i(t)=N^{\mathrm{data}}_i(t) - N^{\mathrm{fit}}_i(t)$，其中 $N^{\mathrm{data}}_i(t)$ 为实验数据在时间 $t$ 处的计数，$N^{\mathrm{fit}}_i(t)$ 为对应的拟合结果。注意到残差的幅度随时间逐渐减小，这主要由于事例数随时间呈指数衰减 $N(t)\sim e^{-t/\tau}$，其泊松涨落满足$\sigma_N \sim \sqrt{N}$，因此残差的典型幅度也随时间减小。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{residual.png}
\caption{模拟结果拟合后残差随时间的分布。}
\end{figure}


随后我对残差进行快速傅里叶变换（FFT），以分析拟合模型未能描述的频率成分。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{fft.png}
\caption{模拟结果残差的快速傅里叶变换（FFT）频谱。橙色为残差频谱，蓝色为原始数据频谱。可以看到残差中主要振荡结构已被拟合函数去除。}
\label{fake_fft}
\end{figure}

可以看到残差的频谱中未出现显著的特征峰，表明拟合后的残差在频域中不包含明显的结构。这说明五参数模型已经能够充分描述模拟数据中的主要时间依赖行为。
