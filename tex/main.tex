\section{引言}

标准模型（Standard Model）是现代物理学迄今为止最成功的理论之一\cite{BOSTON2023}，然而它并非完美无缺：在预测缪子的反常磁矩方面，标准模型的理论计算与实验测量之间存在着显著的差异，这被称为缪子 $g-2$ 问题。在美国费米实验室，缪子反常磁矩的精确测量实验已经进行多年，2021年测量结果显示了与标准模型预测之间的约4.2倍标准差的差距\cite{Li2022Muon}。因此，该实验引发了物理学界的广泛关注和讨论，分析该实验数据的拟合过程对于理解这一现象至关重要。

这个 Toy MC 程序旨在模拟缪子 $g-2$ 实验中的数据生成和分析过程，从模拟数据中提取物理参数，并通过拟合来验证理论模型。同时，我也将把模拟结果与分享的真实实验 ROOT 数据进行比较，以讨论该 Toy 模型在描述真实实验时间谱时的有效性与局限性。

\section{原理与实现}

实验中，极化的缪子被注入一个均匀的磁场中，其圆周运动的频率记作 $\omega_c$，而缪子自旋的进动频率记作 $\omega_s$。缪子的反常磁矩会导致 $\omega_s$ 与$\omega_c$ 之间存在一个差值，计算可得\cite{PhysRevA103}
\begin{equation}
\omega_a = \omega_s - \omega_c= - a_\mu \frac{q}{m_\mu} B
\end{equation}

其中 $a_\mu$ 是缪子的反常磁矩，定义为\cite{BOSTON2023} $a_\mu = \frac{g-2}{2}$。$q$ 是缪子的电荷量，$m_\mu$ 是缪子的质量，$B$ 是磁场强度。通过测量 $\omega_a$ 与 $B$，我们可以间接地测量 $a_\mu$，从而检验标准模型的预测。在真实实验中，磁场强度 $B$ 可以通过质子在储能环中的拉莫进动频率 $\omega_p$ 来实现\cite{Chu2021Muon}，因此 $\omega_a$ 的测量是整个实验的核心。

\subsection{正电子能量分布与计算}

正缪子衰变为一个正电子和两个中微子 $\mu^+ \longmapsto e^+ + \nu_e + \bar{\nu}_\mu$ ，正电子的能量分布由衰变动力学决定。对于极化缪子，正电子概率密度函数由 Michel Spectrum 描述\cite{Fetscher2019MuonDecay}：
\begin{equation}
\frac{d^2 \Gamma}{dx\, d\cos\alpha}
=
\frac{G_F^2 m_\mu^5}{192\pi^3}
\left[
3 - 2x \pm P_\mu \cos\alpha (2x - 1)
\right] \cdot x^2
\end{equation}

在等式左侧是双重微分衰变率，其中 $x = {E_e}/{E_{max}}$ 为电子归一化能量，$\alpha$ 是极化角（正电子射出方向与缪子自旋方向的夹角），$G_F$ 是费米常数，$m_\mu$ 是缪子质量，$P_\mu$ 是缪子极化度。由于实验中使用的是正缪子，因此等式子中的 $\pm$ 取正。记 $\cos \alpha = \Omega$，则简化后正电子的能量分布可以表示为：
\begin{equation}
f(x,\Omega) = x^2 [3 - 2x + P_\mu \Omega (2x - 1)]
\end{equation}

考察缪子衰变的时刻。记缪子参考系为 $\Gamma'$，在该参考系中缪子处于静止状态。以缪子前进方向（动量方向 $\mathbf{p}_\mu$）为 $z$ 轴建立球坐标系，且令缪子自旋 $\mathbf{S}$ 在 $x$-$y$ 平面内的投影为初始轴线。则正电子的动量方向 $\mathbf{p}_e$ 由极角 $\theta$ 和方位角 $\phi$ 描述。此时，记缪子的自旋方向 $\mathbf{S}$ 与 $z$ 轴的夹角为 $\psi$。该角度可以由缪子相位计算得到，在代码中通过 $\psi = \omega_a T + \phi_0$ 计算，其中 $T$ 是缪子存活时间（即缪子产生到此刻的时间），$\phi_0$ 是初始相位。则
\begin{equation}
\Omega = \cos\alpha = \mathbf{S} \cdot \mathbf{p}_e = \sin\psi \sin\theta \cos\phi + \cos\psi \cos\theta
\end{equation}

\begin{figure}[htbp]
\centering
\includegraphics[width=0.8\textwidth]{MuonPositronDiagram_ManimCE_v0.20.1.png}
\caption{缪子衰变示意图，左图为缪子衰变前状态，右图为缪子衰变后正电子状态。图中 $\mathbf{S}$ 是缪子的自旋，$\mathbf{p}_e$ 是正电子的动量。}
\end{figure}

正电子在 $\Gamma'$ 中可以获得的最高能量 $E'_{max} = (m_\mu^2 + m_e^2) / (2 m_\mu)$。 此时正电子能量为 $E'=x E'_{max}$，动量为 $p'=\sqrt{E'^2 - m_e^2}$。根据狭义相对论，正电子在 $\Gamma$ 中的能量 $E$ 可以通过以下公式计算\cite{Chu2021Muon}：
\begin{equation}
E = \gamma (E' + \beta p' \cos\theta )
\end{equation}

%在实验中，我们设定了一个能量阈值 $E_{th}$，只有当 $E_e$ 满足 $E_e > E_{th}$ 时，才会被计数为一个事件。Toy MC 中通过比较计算得到的 $E$ 与 $E_{th}$ 来筛选事件，并将满足条件的事件记录下来用于后续分析。

\section{模拟结果拟合与分析}

\subsection{五参数模型}

在全部模拟事件中，我将满足能量阈值 $E_{th}$ 的事件筛选出来。我首先将时间范围限制在 $[0, 300]\, \mathrm{\mu s}$ 内进行分析。在真实实验中，由于缪子的 Cylclotron period 为\cite{PhysRevD.103.072002} $149.2\, \mathrm{n s}$，所以如果将时间箱宽度设定为这一值可以过滤掉 Fast Rotation 相关的频率成分\cite{BOSTON2023}，从而更清晰地展示反常自旋进动引起的振荡结构。因此为了与真实实验数据的处理方式保持一致，我将时间箱宽度设定为 $\Delta t = 142.9\, \mathrm{n s}$，并对时间进行装箱统计，得到每个时间箱内满足能量阈值的正电子数目。最终得到的结果如图\ref{wiggle_line_fit_plot}所示。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{FAKE300.0.png}
\caption{正电子数目随时间变化图像。横坐标为时间 $T(\mathrm{\mu s})$，纵坐标为单个时间箱内满足能量阈值的正电子数目。其中$T\in[0, 300]\, \mathrm{\mu s}$，$\Delta t = 142.9\, \mathrm{n s}$，$n=4912$。}
\label{wiggle_line_fit_plot}
\end{figure}

从图像可以非常清晰地看到，正电子数目随时间呈现出明显的振荡现象。该图像可以用一个五参数函数\cite{Fienberg2019}进行拟合：
\begin{equation}
  N(t) = N_0 e^{-t/\tau} \left(1 + A\cos(\omega_a t + \phi_0)\right)
  \label{wiggle_fit_function}
\end{equation}

五个需要拟合的参数分别为：初始事件数目 $N_0$，衰减时间常数 $\tau$，振荡幅度 $A$，反常自旋进动频率 $\omega_a$，以及初始相位 $\phi_0$。拟合结果为 $\omega_a = 1.43899(2)\,\mathrm{rad\ \mu s^{-1}}$，$\phi_0 =  0.004(2)\,\mathrm{rad}$，$\tau = 64.42(2)\,\mathrm{\mu s}$。注意到 $\omega_a$ 的注入参数与拟合参数的偏差在 $1\sigma$ 以内，表明五参数模型能够很好地描述数据。

\begin{figure}[htbp]

\caption{图\ref{wiggle_line_fit_plot}的拟合结果展示。红色曲线为拟合结果，蓝色点为生成数据。}
\end{figure}

\subsection{增大时间区间的拟合表现}

在真实实验数据的分析中可以观察到， wiggle plot 通常覆盖多个振荡周期，其有效时间范围约为 $600\text{--}700\,\mathrm{\mu s}$\cite{Fienberg2019}。为检验拟合结果对时间窗口的稳定性，我将分析时间范围扩展至 $600\,\mathrm{\mu s}$ 进行重新拟合，得到 $\omega_a = 1.43897(2)\,\mathrm{rad\ \mu s^{-1}}$。这说明拟合结果对时间窗口的稳定性较好。然而，绘制得到的 wiggle plot 在视觉上出现了明显差异。

\subsection{残差频谱绘制}

拟合完成后，为了研究模型未能描述的时间结构，我构造了数据的绝对残差\cite{BOSTON2023} $r_i(t)=N^{\mathrm{data}}_i(t) - N^{\mathrm{fit}}_i(t)$，其中 $N^{\mathrm{data}}_i(t)$ 为实验数据在时间 $t$ 处的计数，$N^{\mathrm{fit}}_i(t)$ 为对应的拟合结果。

\begin{figure}[h]
\centering
\includegraphics[width=\textwidth]{FAKE_RES600.0.png}
\caption{模拟结果拟合后残差随时间的分布。由于事例数随时间呈指数衰减 $N(t)\sim e^{-t/\tau}$，其泊松涨落满足$\sigma_N \sim \sqrt{N}$，因此残差的典型幅度也随时间减小。}
\label{residual}
\end{figure}

随后我对残差进行快速傅里叶变换（FFT），以分析拟合模型未能描述的频率成分。

\begin{figure}[h]
\centering
\includegraphics[width=\textwidth]{FAKE_FFT600.0.png}
\caption{模拟结果残差的快速傅里叶变换（FFT）频谱。橙色为残差频谱，蓝色为原始数据频谱。}
\label{fake_fft}
\end{figure}

值得注意的是，如图\ref{fake_fft}所示，残差频谱中出现了位于 $f=0.525\,\mathrm{MHz}$ 的强特征峰，以及极其完美的整数倍高次谐波（$2f=1.050\,\mathrm{MHz}$, $3f=1.575\,\mathrm{MHz}$ 等）。经过计算发现，这是由于数据精度截断引入的“假信号”。

在生成模拟数据时，考虑到数据存储限制，我将衰变时间 $T(\mu\mathrm{s})$ 人为截断保留至小数点后两位（见表\ref{event_sample_table}）。在数据分析中，我尝试将时间箱的宽度设定为缪子的回旋周期 $149.2\,\mathrm{ns}$。由于直方图的箱子总数必须是整数，程序进行了取整处理：即 $n = \mathrm{int}(600\,\mu\mathrm{s} / 0.1492\,\mu\mathrm{s}) = 4021$ 个。这就导致实际划分出来的时间箱宽度变为了 $\Delta t = 600\,\mu\mathrm{s} / 4021 \approx 149.217\,\mathrm{ns}$。

将 $10\,\mathrm{ns}$ 的离散数据点划入宽度为 $149.217\,\mathrm{ns}$ 的连续时间箱中，单个时间箱理论上应包含 $14.9217$ 个点。但实际上，单个时间箱内只能包含整数个点。这种小数部分（$0.9217$）的不断累积，在数据直方图上制造出了周期性的锯齿状波动，其基准频率为 $f = \frac{1 - 0.9217}{0.149217\,\mu\mathrm{s}} \approx 0.525\,\mathrm{MHz}$。这一理论计算值与频谱中观察到的异常峰吻合，而其整数倍的高次谐波，正是这种锯齿状波形在频域上的标准特征。

在排除了这个由于数据处理细节造成的假信号之后，残差频谱中未再出现其他显著的物理特征峰。这说明五参数模型已经能够描述模拟数据中的时间依赖行为。

图中可以看到，真实实验数据在整个时间范围内仍然保持了清晰的振荡结构。与 Toy MC 相比，实验数据的统计量更大，因此即使在较晚时间区域， wiggle plot 依然具有较好的连续性。实验数据中高于能量阈值的正电子总数目有 $48,341,327,153$ 个，为模拟样本量的 $4395$ 倍。正是由于这种显著更高的统计量，实验数据在 $T>600\,\mathrm{\mu s}$ 时仍未像模拟结果那样出现明显的分层现象，而是在对数坐标下保持较为平滑的振荡行为。

\subsection{实验数据的残差与频谱分析}

在完成拟合之后，我与模拟部分一样，构造了实验数据相对于拟合函数的绝对残差。

\begin{figure}[H]
\centering
\includegraphics[width=\textwidth]{REAL_RES.png}
\caption{真实实验数据拟合后残差随时间的分布。}
\label{real_residual}
\end{figure}

从图\ref{real_residual}可以看出，实验数据的残差在早时间区域具有较大的波动，而在较晚时间区域逐渐减小并收敛到零附近，这一趋势与总计数随时间指数衰减的行为一致。整体而言，残差围绕零上下分布，没有表现出明显的长期漂移，说明五参数模型已经提取出了实验数据中的主导 $g-2$ 振荡成分。

不过，残差中仍然可以看到较为明显的周期性结构，尤其在早时间到中等时间区域表现得更为清楚。这表明在五参数模型所描述的指数衰减与主导进动信号之外，实验数据中仍然存在额外的时间相关结构。

为了进一步考察这些结构，我分别对原始时间谱和拟合残差进行了快速傅里叶变换（FFT），结果如图\ref{real_fft}所示。

\begin{figure}[H]
\centering
\includegraphics[width=\textwidth]{REAL_FFT.png}
\caption{真实实验数据原始时间谱与拟合残差的快速傅里叶变换（FFT）频谱。蓝线为原始时间谱的 FFT，橙线为残差的 FFT。}

\subsection{模拟与真实实验差异的可能来源}

综合前述结果，Toy MC 与真实实验之间的差异主要可能来自以下几个方面。

首先，统计量存在显著差别。真实实验数据的事例数远高于 Toy MC，这使得实验数据在长时间端仍能保持平滑的时间谱，而模拟结果则更容易受到有限统计的影响，表现出明显的分层与离散化。

其次，Toy MC 的物理模型仍然是理想化的。当前模拟主要包含了 Michel Spectrum 、自旋进动、洛伦兹变换和能量阈值筛选，这已经足以产生标准的 wiggle plot，但尚未纳入真实实验中的探测器几何接受度、束流横向分布、不同量能器之间的响应差异以及更复杂的时间依赖修正。

最后，真实实验数据中还包含束流动力学与探测器系统效应。正式的 $\omega_a$ 分析并不只依赖最简单的五参数函数，而需要进一步考虑诸如 CBO、增益漂移、pileup、缪子损失以及相位接受度等修正项\cite{PhysRevD.103.072002,PhysRevLett.131.161802,Chu2021Muon}。这些效应会在残差和频谱中留下 Toy MC 中不存在的结构，因此也是造成两者差异的重要原因。

综上所述，Toy MC 与真实实验数据之间既有一致，也有差异。其中，一致性体现为对主导进动频率与衰减寿命的良好复现；而差异性则主要反映了真实实验比理想化模拟包含更多统计和系统效应。这种结果是符合预期的，也说明 Toy MC 适合作为理解 wiggle plot 基本形成机制的第一步工具，但不足以完全替代正式实验分析。

\section{总结}

本文构建了一套用于模拟 muon $g-2$ 实验的 Toy Monte Carlo 程序。该程序从极化缪子衰变的 Michel Spectrum 出发，在缪子静止系中生成正电子的能量与角分布，并结合缪子自旋进动和洛伦兹变换，得到实验室系下正电子的时间、能量、动量分量与衰变位置信息。基于这些事件级数据，我进一步构造了与实验中相似的 wiggle plot，并利用五参数函数对其进行了拟合，验证了该模型能够准确回收注入的反常进动频率 $\omega_a$。

在此基础上，我对真实实验 ROOT 数据进行了与模拟部分尽可能一致的处理，包括能量阈值选择、时间装箱、五参数拟合、残差计算以及残差频谱分析。比较结果表明，Toy MC 与真实实验数据在 $\omega_a$ 和 $\tau$ 两个关键参数上符合良好，说明模型已经重现了主导时间结构；同时，两者在振幅、相位和残差频谱细节上仍存在明显差异，这些差异主要归因于统计量差别以及真实实验中更复杂的束流动力学和探测器系统效应。

总体而言，该基于基本衰变动力学与自旋进动的简化 Toy MC 程序，已经能够复现缪子 $g-2$ 实验时间谱的主要特征，并为理解 $\omega_a$ 的提取过程提供直观的物理图像。但若要进一步逼近正式实验分析，还需要在后续工作中引入更完整的探测器响应、束流动力学修正以及更细致的频谱分析方法，从而更好地刻画真实数据中除主导振荡外的次级时间结构。
