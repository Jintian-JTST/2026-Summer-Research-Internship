\section{引言}

标准模型是现代物理学迄今为止最成功的理论之一\cite{BOSTON2023}。然而，它在预测缪子的反常磁矩方面仍存在显著偏差，这一问题通常被称为缪子 $g-2$ 问题。在美国费米实验室，缪子反常磁矩的精确测量实验已经进行了多年。2021 年的测量结果显示，其与标准模型预测之间存在约 $4.2\sigma$ 的差距\cite{Li2022Muon}。随着 2025 年测量精度达到 $127$ ppb，该测量仍在持续推进\cite{7clf-sm2v}。因此，该实验持续引发物理学界的广泛关注和讨论。

本项目中的 Toy MC 程序旨在模拟缪子 $g-2$ 实验中的数据生成和分析过程，从模拟数据中提取物理参数，并通过拟合验证理论模型。同时，我也将把模拟结果与分享的真实实验 ROOT 数据进行比较，以讨论该 Toy 模型在描述真实实验时间谱时的有效性与局限性。

\section{原理与实现}

实验中，极化缪子被注入一个均匀磁场中，其圆周运动频率记作 $\omega_c$，而缪子自旋的进动频率记作 $\omega_s$。缪子的反常磁矩会导致 $\omega_s$ 与 $\omega_c$ 之间出现差值，计算可得\cite{PhysRevA103}
\begin{equation}
\omega_a = \omega_s - \omega_c = - a_\mu \frac{q}{m_\mu} B
\end{equation}

其中 $a_\mu$ 是缪子的反常磁矩，定义为\cite{BOSTON2023} $ a_\mu = \frac{g-2}{2}$；$q$ 是缪子的电荷量，$m_\mu$ 是缪子的质量，$B$ 是磁场强度。通过测量 $\omega_a$ 与 $B$，可以间接测得 $a_\mu$，从而检验标准模型的预测。在真实实验中，磁场强度 $B$ 可以通过质子在储能环中的拉莫进动频率 $\omega_p$ 来实现\cite{Chu2021Muon}，因此 $\omega_a$ 的测量是整个实验的核心。

\subsection{正电子能量分布与计算}

正缪子衰变为一个正电子和两个中微子 $\mu^+ \to e^+ + \nu_e + \bar{\nu}_\mu$，正电子的能量分布由衰变动力学决定。对于极化缪子，正电子概率密度函数由 Michel Spectrum 描述\cite{Fetscher2019MuonDecay}：
\begin{equation}
\frac{d^2 \Gamma}{dx\, d\cos\alpha}
=
\frac{G_F^2 m_\mu^5}{192\pi^3}
\left[
3 - 2x \pm P_\mu \cos\alpha\,(2x - 1)
\right] x^2
\end{equation}

在等式左侧是双重微分衰变率，其中 $x = E_e/E_{max}$ 为电子归一化能量，$\alpha$ 是极化角，即正电子射出方向与缪子自旋方向的夹角，$G_F$ 是费米常数，$m_\mu$ 是缪子质量，$P_\mu$ 是缪子极化度。由于实验中使用的是正缪子，因此式中的 $\pm$ 取正。记 $\cos\alpha = \Omega$，则简化后正电子的能量分布可以表示为：
\begin{equation}
f(x,\Omega) = x^2 \bigl[3 - 2x + P_\mu \Omega (2x - 1)\bigr]
\end{equation}

接下来考察缪子衰变的时刻。记缪子参考系为 $\Gamma'$，在该参考系中缪子处于静止状态。以缪子前进方向（动量方向 $\mathbf{p}_\mu$）为 $z$ 轴建立球坐标系，并令缪子自旋 $\mathbf{S}$ 在 $x$-$y$ 平面内的投影为初始轴线。则正电子的动量方向 $\mathbf{p}_e$ 由极角 $\theta$ 和方位角 $\phi$ 描述。此时，记缪子自旋方向 $\mathbf{S}$ 与 $z$ 轴的夹角为 $\psi$。该角度可以由缪子相位计算得到，在代码中通过 $\psi = \omega_a T + \phi_0$ 计算，其中 $T$ 是缪子存活时间（即缪子产生到此刻的时间），$\phi_0$ 是初始相位。于是有
\begin{equation}
\Omega = \cos\alpha = \mathbf{S}\cdot \mathbf{p}_e = \sin\psi\,\sin\theta\,\cos\phi + \cos\psi\,\cos\theta.
\end{equation}

\begin{figure}[htbp]
\centering
\includegraphics[width=0.8\textwidth]{MuonPositronDiagram_ManimCE_v0.20.1.png}
\caption{缪子衰变示意图，左图为缪子衰变前状态，右图为衰变后正电子状态。图中 $\mathbf{S}$ 是缪子的自旋，$\mathbf{p}_e$ 是正电子的动量。}
\end{figure}

正电子在 $\Gamma'$ 中可以获得的最高能量为 $E'_{max} = (m_\mu^2 + m_e^2)/(2m_\mu)$。此时正电子能量为 $E' = xE'_{max}$，动量为 $p' = \sqrt{E'^2 - m_e^2}$。根据狭义相对论，正电子在实验室系 $\Gamma$ 中的能量 $E$ 可以通过以下公式计算\cite{Chu2021Muon}：
\begin{equation}
E = \gamma\,(E' + \beta p' \cos\theta)
\end{equation}

\section{模拟结果拟合与分析}

\subsection{五参数模型}

在全部模拟事件中，我将满足能量阈值 $E_{th}$ 的事件筛选出来，随后时间范围限制在 $300\,\mathrm{\mu s}$ 内进行分析。在真实实验中，由于缪子的回旋周期约为\cite{PhysRevD.103.072002} $149.2\,\mathrm{ns}$，如果将时间箱宽度设定为这一值，可以过滤掉 Fast Rotation 相关的频率成分\cite{BOSTON2023}，从而更清晰地展示反常自旋进动引起的振荡结构。为了与真实实验数据的处理方式保持一致，我将时间箱宽度设定为 $\Delta t = 149.2\,\mathrm{ns}$，并对时间进行装箱统计，得到每个时间箱内满足能量阈值的正电子数目。此外，为确保时间箱的完整性，我取时间范围为整数个回旋周期，将时间范围 $300\,\mathrm{\mu s}$ 修正为 $300.0412\,\mathrm{\mu s}$。最终结果如图\ref{wiggle_line_fit_plot}所示。

\begin{figure}[htbp]
\centering
\includegraphics[width=\textwidth]{FAKE300.0412.png}
\caption{正电子数目随时间变化图像。横坐标为时间 $T(\mathrm{\mu s})$，纵坐标为单个时间箱内满足能量阈值的正电子数目。其中 $T\in[0,300.0412]\,\mathrm{\mu s}$，$\Delta t = 149.2\,\mathrm{ns}$，$n=2011$。在该时间范围内，高于能量阈值的正电子总数目为 $10{,}893{,}931$ 个。}
\label{wiggle_line_fit_plot}
\end{figure}

从图像可以清晰看到，正电子数目随时间呈现出明显的振荡现象。该图像可以用一个五参数函数\cite{Fienberg2019}进行拟合：
\begin{equation}
  N(t) = N_0 e^{-t/\tau} \left(1 + A\cos(\omega_a t + \phi_0)\right)
  \label{wiggle_fit_function}
\end{equation}

五个需要拟合的参数分别为：初始事件数目 $N_0$、衰减时间常数 $\tau$、振荡幅度 $A$、反常自旋进动频率 $\omega_a$ 以及初始相位 $\phi_0$。拟合结果为 $\omega_a = 1.439007(22)\,\mathrm{rad\,\mu s^{-1}}$，$\phi_0 = 0.0001(18)\,\mathrm{rad}$，$\tau = 64.357(19)\,\mathrm{\mu s}$。注意到 $\omega_a$ 的设定值与拟合结果的偏差在 $1\sigma$ 以内，表明五参数模型能够很好地描述数据。

\begin{figure}[htbp]

\caption{图\ref{wiggle_line_fit_plot}的拟合结果展示。红色曲线为拟合结果，蓝色点为生成数据。}
\end{figure}

在相关文献中\cite{PhysRevD.103.072002,Chu2021Muon,PhysRevLett.131.161802}，常使用 wiggle plot 来展示超过能量阈值的正电子计数随时间的振荡行为。通过将时间对一个固定周期（本文选取 $100\,\mathrm{\mu s}$）取模，可以将长时间范围内的数据折叠，从而更清晰地展示振荡结构。

\subsection{增大时间区间的拟合表现}

在真实实验数据的分析中可以观察到，wiggle plot 通常覆盖多个振荡周期，其有效时间范围约为 $600\text{--}700\,\mathrm{\mu s}$\cite{Fienberg2019}。为检验拟合结果对时间窗口的稳定性，将分析时间范围扩展至 $600.0824\,\mathrm{\mu s}$ 进行重新拟合，得到 $\omega_a = 1.439012(20)\,\mathrm{rad\,\mu s^{-1}}$。这说明拟合结果对时间窗口具有较好的稳定性。然而，绘制得到的 wiggle plot 在视觉上出现了明显差异。

\subsection{残差频谱绘制}

拟合完成后，为了研究模型未能描述的时间结构，我构造了拟合残差\cite{BOSTON2023}
\begin{equation}
r_i(t)=N^{\mathrm{data}}_i(t)-N^{\mathrm{fit}}_i(t),
\end{equation}
其中 $N^{\mathrm{data}}_i(t)$ 为数据在时间 $t$ 处的计数，$N^{\mathrm{fit}}_i(t)$ 为对应的拟合结果。

\begin{figure}[h]
\centering
\includegraphics[width=\textwidth]{FAKE_RES600.0824.png}
\caption{模拟结果拟合后残差随时间的分布。由于事例数随时间呈指数衰减 $N(t)\sim e^{-t/\tau}$，其泊松涨落满足 $\sigma_N\sim\sqrt{N}$，因此残差的典型幅度也随时间减小。}
\label{residual}
\end{figure}

随后我对残差进行快速傅里叶变换（FFT），以分析拟合模型未能描述的频率成分。

\begin{figure}[h]
\centering
\includegraphics[width=\textwidth]{FAKE_FFT600.0824.png}
\caption{模拟结果残差的快速傅里叶变换（FFT）频谱。橙色为残差频谱，蓝色为原始数据频谱。}
\label{fake_fft}
\end{figure}

可以看到残差的频谱中未出现显著的特征峰，表明拟合后的残差在频域中不包含明显的结构。这说明五参数模型已经能够充分描述模拟数据中的主要时间依赖行为。这一结果也为后文与真实实验数据的比较提供了一个理想化参照。

\section{实验数据分析}

为了与 Toy MC 的处理流程保持一致，我对分享的 ROOT 文件中的真实实验数据施加相同的能量阈值条件，并在统一的时间范围内构造正电子计数随时间变化的 wiggle plot。随后，我使用与模拟部分相同的五参数函数式（\ref{wiggle_fit_function}）对实验数据进行拟合，并进一步计算残差及其频谱，用于考察五参数模型对实验数据时间结构的描述程度。

图中可以看到，真实实验数据在整个时间范围内仍然保持了清晰的振荡结构。与 Toy MC 相比，实验数据的统计量更大，因此即使在较晚时间区域，wiggle plot 依然具有较好的连续性。实验数据中高于能量阈值的正电子总数目有 $48{,}341{,}327{,}153$ 个，为模拟样本量的 $4395$ 倍。正是由于这种显著更高的统计量，实验数据在 $T>600\,\mathrm{\mu s}$ 时仍未像模拟结果那样出现明显的分层现象，而是在对数坐标下保持较为平滑的振荡行为。

\subsection{实验数据的残差与频谱分析}

在完成拟合之后，我与模拟部分一样，构造了实验数据相对于拟合函数的绝对残差。

\begin{figure}[H]
\centering
\includegraphics[width=\textwidth]{REAL_RES.png}
\caption{真实实验数据拟合后残差随时间的分布。}
\label{real_residual}
\end{figure}

从图\ref{real_residual}可以看出，实验数据的残差在早时间区域具有较大的波动，而在较晚时间区域逐渐减小并收敛到零附近，这一趋势与总计数随时间指数衰减的行为一致。整体而言，残差围绕零上下分布，没有表现出明显的长期漂移，说明五参数模型已经提取出了实验数据中的主导 $g-2$ 振荡成分。

不过，残差中仍然可以看到较为明显的周期性结构，尤其在早时间到中等时间区域表现得更为清楚。这表明在五参数模型所描述的指数衰减与主导进动信号之外，实验数据中仍然存在额外的时间相关结构。为了进一步考察这些结构，我分别对原始时间谱和拟合残差进行了快速傅里叶变换（FFT），结果如图\ref{real_fft}所示。

\begin{figure}[H]
\centering
\includegraphics[width=\textwidth]{REAL_FFT.png}
\caption{真实实验数据原始时间谱与拟合残差的快速傅里叶变换（FFT）频谱。蓝线为原始时间谱的 FFT，橙线为残差的 FFT。}
\label{real_fft}
\end{figure}

图\ref{real_fft}表明，残差在频域中并非完全平坦，而是仍然保留了若干可辨识的峰结构。这说明在主导的指数衰减和 $g-2$ 振荡之外，实验数据中还存在额外的时间调制成分。由于当前分析仍采用较为简化的总谱处理方法，因此这里的 FFT 更适合被理解为一种定性诊断工具，用来揭示拟合后仍然保留下来的频率结构。

由表\ref{fit_compare_table}可以看出，在反常自旋进动角频率 $\omega_a$ 和衰减寿命 $\tau$ 这两个核心物理量上，Toy MC 的拟合结果与程序设定值高度一致，同时也与真实实验数据的拟合结果较为接近。这说明当前的简化模拟已经能够较好地复现 wiggle plot 的主导时间结构。

相比之下，归一化参数 $N$、振幅参数 $A$ 与初始相位 $\phi_0$ 在模拟与实验之间存在明显差异。这种差异是可以预期的。$N$ 主要受统计量规模影响，$A$ 受能量阈值的影响\cite{BOSTON2023}，而 $\phi_0$ 则主要被探测器接受度、束流初始相空间分布等实验因素影响。由于当前 Toy MC 只包含衰变动力学、自旋进动和阈值筛选等基本物理过程，尚未纳入真实实验中的各种实验细节，因此这些表观参数不必与实验结果完全一致。

由于我使用的是绝对残差而非归一化残差，因此两组数据残差幅度的大小会显著受到总计数规模的影响，不能直接作为模型优劣的定量判据，更值得关注的是残差的内部特征。从图\ref{comp_residual}可以看到，Toy MC 的残差主要表现为围绕零点的随机分布，而真实实验数据的残差在早时间到中等时间区域仍然保留了较明显的规则振荡。这说明真实实验数据中存在 Toy MC 尚未包含的附加时间相关结构。

\subsection{模拟与真实实验差异的可能来源}

尽管 Toy MC 已经能够在主导层面复现实验数据的指数衰减与 $g-2$ 振荡结构，但与真实实验数据相比，二者在残差分布与频谱结构上仍存在明显差异。当前 Toy MC 主要保留了缪子衰变、自旋进动、洛伦兹变换和能量阈值筛选等核心过程，而没有完整纳入真实实验中更复杂的束流动力学效应\cite{PhysRevD.103.072002,PhysRevLett.131.161802,Chu2021Muon}。这些效应会在残差中留下额外的时间结构，并在频域中表现为附加峰值。此外，真实实验数据的事例数远高于 Toy MC，因此在较晚时间区域仍能保留较稳定的振荡结构；而 Toy MC 在统计量较低时残差更接近随机噪声。

\section{总结}

本文构建了一套用于模拟 muon $g-2$ 实验的 Toy Monte Carlo 程序。该程序基于缪子衰变动力学、自旋进动和洛伦兹变换，在事件级别生成了正电子的时间、能量、动量分量与衰变位置信息。基于这些模拟事件，我进一步构造了与实验处理方式一致的 wiggle plot，并利用五参数函数对其进行了拟合，验证了该模型能够准确回收注入的反常进动频率 $\omega_a$。

在此基础上，我将真实实验 ROOT 数据按照相同思路进行了处理，包括能量阈值选择、时间装箱、五参数拟合、残差计算以及残差频谱分析。比较结果表明，Toy MC 与真实实验数据在 $\omega_a$ 和 $\tau$ 这两个关键参数上符合良好，说明该模型已经能够复现 wiggle plot 的主导时间结构；同时，两者在振幅、相位和残差频谱细节上仍存在明显差异，这些差异主要来自真实实验中更复杂的束流动力学、探测器响应和统计效应。

总体而言，该简化 Toy MC 已经能够较好地复现缪子 $g-2$ 实验时间谱的主要特征，并为理解 $\omega_a$ 的提取过程提供了清晰的物理图像。但若要进一步逼近正式实验分析，还需要在后续工作中引入更完整的探测器响应、束流动力学修正以及更细致的频谱分析方法。
